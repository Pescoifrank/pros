About This Center

The Reports & Violation Center allows you to report any activity that violates ethical, legal, or operational guidelines in the fields of:

  • Cyber Security
  • Programming
  • Information Science
  • Software Development
  • Computer Networks & Communication

When to Report

You should file a report if you witness or suspect:

  • Unauthorized access to systems or sensitive data
  • Malicious software or code exploitation
  • Infringement of intellectual property in programming
  • Network misuse or communication breaches
  • Violations of user privacy or data rights
